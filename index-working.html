What is a Convolutional Neural Network?

A Convolutional Neural Network (CNN) is a type of AI that learns to recognize patterns in images, much like our own brain.

Instead of seeing an entire image at once, a CNN scans it piece by piece, looking for simple features like edges and corners. It then combines these features to understand more complex objects.

How CNNs Work: A Step-by-Step Journey

1. Input Image

We begin with a simple image of a handwritten digit. The network sees this image as a grid of pixels.

Filter Size: 3√ó3
Operation: Edge Detection
Output: Feature Map

2. Convolution: Finding Patterns

A small filter (or kernel) slides across the image, searching for basic patterns like edges and curves. Each filter is specialized to find a specific feature.

3. Pooling: Simplifying

Max pooling simplifies the information by taking the most important features from a small area. This makes the network more efficient and robust.

4. Classification: The Final Decision

Finally, a standard neural network takes the simplified features and makes a prediction. It outputs a probability for each digit, and the highest one is the network's guess.

This is a simplified version of the Python code used to build the CNN.

Python (using TensorFlow/Keras) 
import tensorflow as tf
from tensorflow.keras import layers, models

# Build the CNN model
model = models.Sequential([
    layers.Conv2D(32, (3, 3), activation='relu', input_shape=(28, 28, 1)),
    layers.MaxPooling2D((2, 2)),
    layers.Conv2D(64, (3, 3), activation='relu'),
    layers.MaxPooling2D((2, 2)),
    layers.Flatten(),
    layers.Dense(10, activation='softmax')
])

# Compile the model
model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

# Load MNIST data and train the model
(train_images, train_labels), (test_images, test_labels) = tf.keras.datasets.mnist.load_data()
train_images = train_images.reshape((-1, 28, 28, 1)) / 255.0
test_images = test_images.reshape((-1, 28, 28, 1)) / 255.0

model.fit(train_images, train_labels, epochs=5)

# Evaluate the model
test_loss, test_acc = model.evaluate(test_images, test_labels)
print(f"Test accuracy: {test_acc}")
