02 Feature Extraction

Finding Features with Filters

Understanding Convolution

  • Convolution: A filter slides across the image, multiplying and summing values
  • Multiple Filters: Different filters detect different features (edges, curves, etc.)
  • Feature Maps: Each filter creates a feature map highlighting specific patterns
  • Learning: The network learns what filters work best for digit recognition

03 Data Reduction

Simplifying with Pooling

The feature maps are quite large, so we use pooling to reduce their size while keeping the most important information. This makes the network faster and more efficient.

Understanding Pooling

  • Size Reduction: Pooling reduces the spatial dimensions of feature maps
  • Max Pooling: Takes the maximum value from each pooling window
  • Translation Invariance: Makes the network less sensitive to small shifts
  • Efficiency: Fewer parameters means faster training and inference

Understanding Classification

  • Flattening: All feature maps are combined into one long vector
  • Dense Layers: Fully connected layers learn complex patterns
  • Softmax: Final layer converts outputs to probabilities
  • Prediction: Highest probability determines the predicted digit
