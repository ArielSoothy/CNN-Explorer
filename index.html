What is a Convolutional Neural Network?

A Convolutional Neural Network (CNN) is a type of AI that learns to recognize patterns in images, much like our own brain.

Instead of seeing an entire image at once, a CNN scans it piece by piece, looking for simple features like edges and corners. It then combines these features to understand more complex objects.

How CNNs Work: A Step-by-Step Journey

1. The Input Image

Everything starts with an image. The network sees this not as a picture, but as a grid of numbers (pixels), where each number represents a color or intensity.

For this demo, we use a 28x28 pixel image of a handwritten digit.

2. Convolution: Finding Features

A small filter (or kernel) slides over the image. It's a tiny grid of numbers trained to recognize a specific pattern, like an edge or a curve.

The output is a feature map, which highlights where the feature was found.

3. Pooling: Down-sampling

The pooling layer shrinks the feature map, keeping only the most important information. This makes the network faster and more robust to small changes in the image.

Max pooling, a common technique, takes the maximum value from each small block of the feature map.

4. Classification: The Final Guess

After several convolution and pooling layers, the processed features are "flattened" into a single list and fed into a standard neural network.

This final part, the classifier, analyzes the features and makes the final prediction about which digit it is.

Try Different Filter Sizes

Larger filters capture broader patterns but lose fine details. Smaller filters catch edges and tiny features.

Adjust Filter Count

More filters help the network learn more diverse patterns but increase computation time.

See the Code

This is a simplified version of the Python code used to build the CNN.

Python (using TensorFlow/Keras) 
import tensorflow as tf
from tensorflow.keras import layers, models

# Build the CNN model
model = models.Sequential([
    layers.Conv2D(32, (3, 3), activation='relu', input_shape=(28, 28, 1)),
    layers.MaxPooling2D((2, 2)),
    layers.Conv2D(64, (3, 3), activation='relu'),
    layers.MaxPooling2D((2, 2)),
    layers.Flatten(),
    layers.Dense(10, activation='softmax')
])

# Compile the model
model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

# Load MNIST data and train the model
(train_images, train_labels), (test_images, test_labels) = tf.keras.datasets.mnist.load_data()
train_images = train_images.reshape((-1, 28, 28, 1)) / 255.0
test_images = test_images.reshape((-1, 28, 28, 1)) / 255.0

model.fit(train_images, train_labels, epochs=5)

# Evaluate the model
test_loss, test_acc = model.evaluate(test_images, test_labels)
print(f"Test accuracy: {test_acc}")
